## 1. Supports
Javascript, Typescript, JSX

## 2. Download
### 2.1 For Javascript
eslint (https://www.npmjs.com/package/eslint)

### 2.2 For Typescript

## 3. How to Use?
### 3.1 For Javascript
```json
{
  "extends": ["sushi"]
}
```
### 3.2 For Typescript
```json
{
  "extends": ["sushi/typescript"]
}
```
